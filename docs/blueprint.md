# **App Name**: KaliAccess

## Core Features:

- User Registration: Allows new users to register by providing their full name, monthly start date, monthly end date, and a profile picture.
- Webcam Capture: Enables users to take a photo using their webcam during registration.
- Photo Upload: Allows users to upload a profile picture from their device during registration.
- Fingerprint Scan: Requests that new users scan a fingerprint during registration using their computer's scanning device.

## Style Guidelines:

- Primary color: Dark teal (#008080) to reflect a sense of trustworthiness.
- Background color: Light teal (#E0F8F8), a lighter shade of teal.
- Accent color: Forest green (#228B22) for active elements, enhancing the calm tone.
- Font: 'Inter', a sans-serif font that offers a neutral and modern aesthetic, ideal for maintaining readability across the platform. Suitable for both headings and body text.
- Clean and straightforward layout with clear form fields and instructions for ease of use during registration.
- Simple, recognizable icons to represent different user actions and features, such as uploading a photo or scanning a fingerprint.
- Subtle animations for form transitions and interactive elements, providing a smooth and engaging user experience.
